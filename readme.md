# mermaid-схема
## Структура
HEAD -- это голова.<br>
Коммит -- это всему голова.<br>
Статусы файлов:<br>

```mermaid
%% код: пустая строка-три черточки-язык кода-три черточки-пустая строка
%% тип схем (graph LR); стрелка без текста A --> B; между строк ;
graph TD;
      A-->B;
      A-->C;
      B-->D;
      C-->D;
```

## Пример

```mermaid
graph LR;
  untracked -- "git add" --> staged;
  staged    -- "???"     --> tracked/comitted;
```
